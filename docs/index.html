Sobre

The Box foi desenvolvida para ser de alta qualidade, com recursos avançados e uma alternativa acessível em relação as outras impressoras 3D no mercado, para que todos possam ter uma impressora 3D de qualidade. Todos os planos, software e peças são totalmente gratuitos para baixar modificar e fazer o que quiser, desde que você siga as licenças.

Recursos futuros

  • Sistema integrado de nivelamento mecânico automatico, limpeza de bico e mesa.
  • Otimizar o volume de impressão para 22cmx22cmx30cm.
  • Tornar a impressora mais simples para montar.
  • Reduzir o número de parafusos.
  • Mesa aquecida 127/220v.
  • Extrusão dupla avançada.
  • Coletor de extrusão primária.
  • Mesa de impressão removível e magnética.
  • Trocar os rolamentos lineares XY por buchas.
  • Detecção de erros por visão computacional.
  • Centro de controle na cloud para que sua impressora esteja sempre ao alcance.
